# Vectre Labs

# Installation
1. Install [Node.js v18.20 (npm 8.9.0)](https://nodejs.org/en/)
2. Install required dependencies (start from root directory then run the following):
    ```powershell
    cd backend
    npm install
    cd ../frontend
    npm install
    ```
3. Start server
    ```powershell
    cd backend
    npm run
    ```
4. Start server (run in seperate terminal)
    ```powershell
    cd frontend
    npm run
    ```
